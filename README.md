
# Live Smishing Investigation Project

## Overview
This project involves a detailed investigation into a live smishing (SMS phishing) attempt. The goal was to analyze the delivery vector, understand the malicious intent, and document the findings for educational and security research purposes.

## Files Included
- **live_smishing_investigation_project.pdf**: The detailed analysis report.
- **threat_intelligence_report.pdf**: Supporting threat intel data.
- **Evidence images**: Screenshots capturing the attack flow.

## Methodology
- SMS Header Analysis.
- URL/Link inspection.
- Threat Intelligence correlation.

## Disclaimer
All evidence has been sanitized to remove any PII (Personally Identifiable Information). This project is intended for professional portfolio use.
